LaTeX

Never fight with Word to format equations or place figures again!

LaTeX is not a word processor, but a "document preparation system for high-quality typsetting". While it is often the preferred tool for preparing technical reports, scientific papers, and academic journal articles, in part due to the features I describe below, LaTeX can be used to prepare and typeset practically any sort of document, including but not limited to presentations, letters, posters, and books.

I recommend LaTeX beginners start out with an online compiler such as Overleaf, which will allow you to type on one side of the screen and see the compiled document on the other. Compiling offline is also easier than you might think; I can point you to more information about that if you're interested.

Some other online tools I have found valuable while TeXing are as follows:

  • Tables Generator—Paste table data from a spreadsheet and it will spit out the LaTeX code for you.
  • De-TeXify—Draw the symbol you want; it will tell you the LaTeX command and package.
  • LaTeX equation editor—Build and edit your equations graphically, then copy and paste into your code.
  • The LaTeX Stack Exchange—for any and all general questions; someone else has likely already had and solved your problem.
